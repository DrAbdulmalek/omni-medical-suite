# 🏥 معالج الوثائق الطبية — الإصدار النهائي الموحد

## ✨ الميزات
- **قص ذكي** يعتمد على Median فقط (لا يقطع النصوص الكثيفة)
- **كشف الميلان** بالتحقق من الصحة (يمنع الزوايا الخاطئة)
- **معالجة دفعية غير حاجبة** مع شريط تقدم
- **OCR لاستخراج أرقام الصفحات** وإعادة الترتيب تلقائياً
- **نظام تعلّم تكيفي** (KNN) يقترح الإعدادات من المعالجات السابقة
- **تقييم جودة الصورة** (وضوح، تباين، سطوع)
- **دعم PDF** (تحويل كل صفحة إلى صورة)
- **سحب وإفلات** للملفات والمجلدات
- **اختصارات لوحة المفاتيح** (Ctrl+Z, Ctrl+S, الأسهم, F11)

## 🚀 التثبيت والتشغيل

```bash
git clone https://github.com/DrAbdulmalek/medical-doc-processor.git
cd medical-doc-processor
chmod +x install.sh run.sh
./install.sh
./run.sh
```

## 🧪 تشغيل الاختبارات
```bash
source .venv/bin/activate
pytest desktop/test_core.py -v
```

## ⌨️ الاختصارات
| المفتاح | الوظيفة |
|---------|----------|
| `Ctrl+Z` | تراجع |
| `Ctrl+Y` | إعادة |
| `Ctrl+S` | حفظ الصورة الحالية |
| `→` / `←` | التنقل بين الصفحات |
| `Ctrl+D` | كشف الميلان وتصحيحه |
| `Ctrl+G` | قص ذكي |
| `F11` | ملء الشاشة |

## 📦 الاعتماديات
- Python 3.8+
- OpenCV, NumPy, PyQt5
- pytesseract, pdf2image, Pillow
- Tesseract OCR (مع دعم العربية)
- Poppler (لتحويل PDF)
